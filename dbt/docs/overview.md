# 📊 Lab dbt MBA — Sistema de Ponto Eletrônico (Simplificado)

Bem-vindo à documentação do projeto **lab-dbt-mba**!

Este projeto foi construído e simplificado como **guia didático** para alunos de MBA aprendendo
modelagem dimensional e dbt. Ele transforma dados transacionais de um sistema de
ponto eletrônico em um Data Warehouse analítico.

---

## 🎯 Visões Analíticas de Destaque

- **analytic_horas_mensais** — O básico de qualquer RH.
- **analytic_absenteismo_departamento** — KPI focado em negócio.
- **vw_funcionarios_anonimizados** — Exemplo de compliance com LGPD.
